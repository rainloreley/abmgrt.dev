# Privacy Policy for Shlink Manager (Android)

Shlink Manager (for Android) does not collect any user data.<br/>
The only connection to the internet takes place when requesting data from the Shlink instance. In this case, user-provided information gets sent to the server, which was configured upon the first app launch.<br/>
The owner of this instance is responsible for any data. We are not affiliated with Shlink and do not store user data stored on the instance anywhere else except on your local device.<br/>
This project is open source. Feel free to <a className={"underline text-blue-500"} href={"https://github.com/rainloreley/shlink-manager"} target={"_blank"}>check the source code yourself.</a>
